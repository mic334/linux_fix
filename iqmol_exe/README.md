# IQmol Executable

This directory contains the **IQmol executable (`iqmol.exe`)** for Linux/Windows systems.

The executable is the standard version downloaded from the official IQmol website. Users can run it directly without additional installation steps.

---

## Usage

To run IQmol:

1. Navigate to this directory in a terminal or file explorer.
2. Execute the file:

```bash
./iqmol.exe
